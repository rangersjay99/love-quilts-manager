Love Quilts Manager Update 7.7.7
Released: July 22, 2026

Needs editing update for today’s inventory test.

- Planned Needs now edit directly inside the selected Needs card.
- Month, charity, size, quantity, note, and distribution fields all load into the inline editor.
- Save Changes and Cancel stay with the record being edited.
- Mark Distributed opens the same inline editor with the distributed quantity and date prepared for review.
- Delete is smaller and still requires a confirmation before removal.
- Availability is labeled “Available for this need” to clarify that inventory is allocated in month order.
- Keeps the Update 7.7.6 report comparison layout.
- No authentication, Firebase sync, or stored-data format changes.
