# Love Quilts Manager

Current release: Update 7.8.16 — July 23, 2026

## Update 7.8.16 — Report, count, sync, and distribution safeguards

- Report totals, headings, and PDF wording now use the same configurable labels as Home.
- On-screen, printed, compact PDF, and full PDF comparison rows use alternating shading for easier reading.
- A visible **Sync Now** button is available in the shared-sync banner and Settings.
- **Set Current Count** records the exact physically counted inventory for a charity and size, including zero, while preserving an adjustment audit record.
- Automatic distribution inventory transactions are linked and protected from direct editing or deletion; matching 7.8.15 automatic distribution entries are linked during upgrade.
- Increasing a distribution without inventory removal requires confirmation that Quilts Out was already recorded separately.
- Reducing an automatically deducted distribution safely adds the corrected quantity back to inventory after confirmation.
- Requests with protected distribution-linked inventory history cannot be deleted, preserving the audit trail; corrections are made through Mark Distributed.
- Existing inventory, requests, reports, settings, recovery history, and Firebase synchronization are retained.
