# Love Quilts Manager

Current release: Update 7.8.17 — July 24, 2026

## Update 7.8.17 — Yearly statistics, Home charity summary, and calendar refinements

- Adds **This Year** totals on Home for quilts made and quilts distributed.
- Adds a direct **By Charity** Home summary showing Available in Storage, Quilts Still Needed, and More to Make.
- Adds selectable **Yearly Statistics** in Reports and exported PDFs: quilts made, quilts distributed, net inventory change, current inventory, charities served, and lifetime quilts distributed.
- Fixes wrapping on the third Reports summary block so long wording is not cut off.
- Shades past calendar months light green when demand was met and light red when it was not met.
- Keeps future months green when covered early and gives the current month a bold border.
- Uses clearer **Available in Storage** wording where appropriate.
- Retains all Update 7.8.16 report, Sync Now, Set Current Count, zero-balance, audit, and distribution safeguards.
- Existing inventory, requests, reports, settings, recovery history, and Firebase synchronization are retained.
