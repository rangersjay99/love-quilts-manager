# Love Quilts Manager

Current release: Update 7.7.2 — July 20, 2026

## Update 7.7.2 — Distributed and completed needs

- Records Quantity Distributed and Date Distributed for each planned need.
- Shows completed needs as **Distributed** with Need, Sent, and Remaining totals.
- Shows unfinished needs from past months as **Past Due** rather than Covered.
- Removes completed needs from current-inventory allocation so old completed months cannot consume today's stock.
- Supports partial distributions.
- Can optionally create a matching Inventory Out transaction for newly distributed quilts.
- Leaves inventory unchanged when the distribution was already recorded manually.
- Keeps the direct quantity-entry fix and approved-device Firebase sync.

The Firebase production organization path and existing stored data remain compatible.
