## Update 7.8.18 — Verified Home, Calendar, and Reports refinement

- Places the **Calendar first**, immediately after the Home **At a Glance** summary.
- Retains the agreed Calendar indicators:
  - Past demand met: light green.
  - Past demand not met: light red.
  - Future request covered early: light green.
  - Current month: noticeably bold border.
- Uses the verified wording consistently on Home and Reports:
  - **Available in Storage**
  - **Quilts Requested**
  - **Quilts Needed to be Completed**
- Migrates the app's older default wording to the verified wording while preserving genuinely customized labels.
- Makes the third Reports summary block span the full row on iPhone so the wording cannot be cut off.
- Calculates **Quilts Needed to be Completed** by charity and size, preventing surplus in one category from hiding a shortage in another.
- Includes the wording and calculation in on-screen reports, printable reports, and both PDF formats.
- Retains yearly statistics, direct Home actions, direct Calendar add/edit, Sync Now, Set Current Count, report row shading, zero-balance handling, audit history, and distribution safeguards.
- Includes `MASTER_CHANGE_LIST.txt` as the written source of truth for future sessions.
