# Love Quilts Manager

Current release: Update 7.8.10 — July 23, 2026

## Update 7.8.10 — At a Glance action shortcuts and clearer difference status

- Adds a **＋ Add Need** shortcut to the Quilts Still Needed card.
- Adds a **Distribute Quilts** shortcut to the Difference card.
- Tapping the Still Needed card opens the detailed charity-request list.
- Tapping the Difference card opens the distribution workflow.
- The main Difference number no longer uses a minus sign.
- A negative difference is shown as a bold red **Shortage**; a positive difference is shown as a bold green **Surplus**; zero shows **Balanced**.
- Retains shared customizable Home wording, the corrected Reports summary, calendar add/edit tools, Firebase synchronization, and existing data.
