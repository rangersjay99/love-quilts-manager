# Love Quilts Manager

**Current release: Update 7.2 — July 15, 2026**

Love Quilts Manager is a simple mobile-friendly inventory and needs-planning app for Faithful Circle Quilters.

## Main features

- Track quilts in, quilts out, and inventory adjustments
- View total quilts on hand and upcoming needs
- Plan monthly needs by charity and quilt size
- Flag adjusted transactions in history and reports
- Export backups, transaction CSV files, and printable reports
- Customize the organization name, charities, and quilt sizes

## Default charities

Grassroots, SHP, St. Agnes, Bridges, and Project Holiday.

## Default sizes

Children's Large, Adult Large, and Medium.

## Installation

Upload all files and the complete `icons` folder to the GitHub repository. Netlify will publish the changes from GitHub.
