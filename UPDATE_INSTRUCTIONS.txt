LOVE QUILTS MANAGER — UPDATE 7.8.14

BEFORE UPDATING
1. Confirm the app says all changes are synced.
2. Export a current backup from Settings.
3. Download a backup ZIP of the current GitHub repository.

UPLOAD
1. Extract this ZIP.
2. Upload every extracted file to the GitHub repository main/root folder.
3. Replace files with the same names.
4. Do not delete or replace the firebase-test folder.
5. Wait for GitHub Pages to publish.
6. Close and reopen the app on each device. Refresh once if an older version remains.

This update does not clear or reinitialize shared inventory.
