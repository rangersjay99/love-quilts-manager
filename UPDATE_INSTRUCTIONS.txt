LOVE QUILTS MANAGER — UPDATE 7.6.0
Firebase Production Live Sync

UPLOAD
1. Extract this ZIP.
2. In the GitHub repository, open the main/root folder.
3. Upload every extracted file and replace files with the same names.
4. Wait for GitHub Pages to publish.
5. Open the regular Love Quilts Manager address and refresh.

FIRST PRODUCTION SETUP
1. Sign in with an approved Firebase email and password.
2. Choose ONE primary device.
3. Confirm its current local data is the desired starting data.
4. Clear fake data with Settings > Start Fresh for Real Use when appropriate.
5. In Shared Device Sync, choose Create Real Shared Inventory.
6. Type START SHARED INVENTORY exactly.
7. Open the second device and sign in. It should receive the shared data.

IMPORTANT
- Do not initialize production from both devices.
- firebase-test remains separate.
- Both accounts normally remain signed in on their own devices.
