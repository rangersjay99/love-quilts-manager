LOVE QUILTS MANAGER — UPDATE 7.8.16
Report Consistency, Exact Counts, Sync Now, and Distribution Safeguards

BEFORE UPDATING
1. Confirm the current device shows all changes synced.
2. Export a current backup from Settings.
3. Keep a ZIP backup of the current GitHub repository.

UPLOAD
1. Extract this ZIP.
2. Upload every extracted file to the main/root folder of the GitHub repository.
3. Replace files with the same names.
4. Do not delete or replace the firebase-test folder.
5. Wait for GitHub Pages to publish.
6. Close and reopen the app on each device. Refresh once if the old version remains.

VERIFY
1. Confirm the splash and About section show Update 7.8.16.
2. Tap Sync Now and wait for “Shared data verified · All changes synced.”
3. Open Reports and confirm wording matches Home and rows alternate shading.
4. Test Set Current Count with a known charity and size.
5. Test one small distribution and confirm the inventory balance changes exactly once.

IMPORTANT
- Set Current Count is intended for a physical recount. It records the difference as an adjusted transaction so the audit history remains intact.
- Leave distribution inventory removal unchecked only when the same quilts were already entered separately as Quilts Out.
- Linked automatic distribution transactions are intentionally protected; correct them through Mark Distributed.
- This update does not clear or reinitialize shared data.
