LOVE QUILTS MANAGER — UPDATE 7.7.6

WHAT THIS UPDATE CHANGES
- Report summary now shows Charity, Size, Requested Needs, On Hand, and Difference.
- On Hand is bold.
- Only Difference is color-coded: green for positive, red for negative, neutral for zero.
- The updated summary is included on screen, in Print Full Report, Print Compact Report, Full PDF, and Compact PDF.

HOW TO UPDATE ON GITHUB
1. Open the faithful-circle-quilt-guild repository.
2. Choose Add file, then Upload files.
3. Upload every file and folder from this ZIP.
4. Allow replacement of the existing files.
5. Commit the changes.
6. Wait about one minute, then reopen or refresh the app.
7. Confirm the splash screen shows Update 7.7.6.

DATA SAFETY
This update does not clear or convert shared data. Firebase login and synchronization are unchanged.
