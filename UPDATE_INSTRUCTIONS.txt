LOVE QUILTS MANAGER — UPDATE 7.1
=================================

This corrected package fixes both items reported after Update 7:

- Restores the colorful patchwork-heart Love Quilts logo
- Uses new icon filenames so the old cached phone icon is not reused
- Adds an updated README.md for the GitHub home page
- Updates the service-worker cache to Update 7.1
- Keeps the same app data storage key so existing records remain

GITHUB UPDATE
-------------
1. Open the GitHub repository.
2. Choose Add file > Upload files.
3. Upload ALL files from this ZIP, including README.md and the complete icons folder.
4. Commit the changes.
5. Wait for Netlify to finish deploying.

PHONE ICON
----------
After Netlify finishes, open the website once in Safari or Chrome and refresh it.

If the old home-screen icon still appears, remove only the old home-screen shortcut and add the app to the home screen again. Export a backup first as an extra precaution. Removing the shortcut does not change the GitHub or Netlify files.
