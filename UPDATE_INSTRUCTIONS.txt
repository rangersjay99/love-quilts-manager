LOVE QUILTS MANAGER — UPDATE 7.5
=================================

UPLOAD
------
1. Open the GitHub repository.
2. Choose Add file > Upload files.
3. Upload ALL files from this ZIP, including GOOGLE_BACKUP_SETUP.txt and the complete icons folder.
4. Replace files with the same names and commit the changes.
5. Wait for GitHub Pages to publish.

IPHONE REFRESH
--------------
1. Open the website in Safari.
2. Refresh the page.
3. The welcome screen should show Update 7.5.
4. Do not clear Safari website data. Existing Update 7.4 records use the same storage key and remain available.
