LOVE QUILTS MANAGER — UPDATE 7.4
=================================

This package restores the detailed full report while keeping the compact report:

- Download Full PDF Report is now the main report button
- The full PDF includes all inventory, upcoming needs, and adjusted transactions
- The full report may use multiple pages so information is not omitted
- Download Compact One-Page PDF remains available for quick meetings
- Full and compact print buttons are both retained
- Generated PDFs do not include the website address
- Update 7.3 customizable names and wording are retained
- Existing inventory and needs are preserved because the storage key remains love_quilts_v1

GITHUB UPDATE
-------------
1. Open the GitHub repository.
2. Choose Add file > Upload files.
3. Upload ALL files from this ZIP, including README.md and the complete icons folder.
4. Choose to replace files with the same names.
5. Commit the changes.
6. Wait for GitHub Pages to finish publishing.

IPHONE REFRESH
--------------
1. Open the website in Safari after GitHub Pages publishes it.
2. Refresh the page.
3. The welcome screen should show “Update 7.4.”
4. Open Reports to choose the full or compact report.

Do not clear Safari website data because that could remove locally stored app records.
