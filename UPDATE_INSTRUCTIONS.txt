LOVE QUILTS MANAGER — UPDATE 7.7.3
Login-Loop Hotfix

BEFORE UPLOADING
1. Do not clear browser/app data.
2. Do not choose Create Real Shared Inventory.
3. If possible, keep the current repository ZIP as a backup.

UPLOAD
1. Extract this ZIP.
2. Open the main/root folder of the GitHub repository.
3. Upload every extracted file and replace files with the same names.
4. Leave the firebase-test folder unchanged.
5. Wait for GitHub Pages to finish publishing.

AFTER UPLOADING
1. Completely close the installed app or browser tab.
2. Reopen the normal Love Quilts Manager address while connected to the internet.
3. The splash screen should show Update 7.7.3.
4. The approved account should remain signed in or allow a normal sign-in without looping.
5. Confirm the header reaches All changes synced.

This hotfix does not reset or reinitialize local or Firebase data.
