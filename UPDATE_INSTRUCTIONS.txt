LOVE QUILTS MANAGER — UPDATE 7.7.4
Distributed Needs in Reports

BEFORE UPLOADING
1. Confirm the app shows All changes synced.
2. Export a current app backup.
3. Keep the current repository ZIP as a backup.
4. Do not choose Create Real Shared Inventory.

UPLOAD
1. Extract this ZIP.
2. Open the main/root folder of the GitHub repository.
3. Upload every extracted file and replace files with the same names.
4. Leave the firebase-test folder unchanged.
5. Wait for GitHub Pages to finish publishing.

AFTER UPLOADING
1. Completely close and reopen the app while connected to the internet.
2. Confirm the splash screen shows Update 7.7.4.
3. Confirm the approved account signs in normally and reaches All changes synced.
4. Open Reports and verify the Distributed Needs section.
5. Check both the full and compact report previews or PDFs.

This update does not reset, reinitialize, or migrate local or Firebase data.
