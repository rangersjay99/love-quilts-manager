LOVE QUILTS MANAGER — UPDATE 7.2
=================================

This package adds the requested report totals and confirmation pop-ups:

- Inventory On Hand report is grouped by charity
- Each charity has its own total
- The report ends with a Grand Total
- Quilts Out requires an “Are you sure?” confirmation before saving
- Inventory adjustments require an “Are you sure?” confirmation before saving
- Confirmations show current inventory, new inventory, and total quilts on hand before and after the change
- Existing inventory, needs, settings, and logo are preserved

GITHUB UPDATE
-------------
1. Open the GitHub repository.
2. Choose Add file > Upload files.
3. Upload ALL files from this ZIP, including README.md and the complete icons folder.
4. Choose to replace files with the same names.
5. Commit the changes.
6. Wait for Netlify to finish deploying and show the new deployment as Published.

IPHONE REFRESH
--------------
Open the website in Safari after Netlify publishes it and refresh the page.
The app should show “Update 7.2” on the welcome screen.

Do not clear Safari website data because that could remove locally stored app records.
